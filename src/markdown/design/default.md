# 插画占位及人物比例

根据缺省页具体主题，拼接出需要的场景，人物主体根据主题定义颜色，其余元素以灰色为主

插画应处于红色虚线范围(462\*268px）内并竖向居中放置，人物比例应控制在插画的 20%左右
